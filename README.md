### 🚀 Featured Projects

#### 🧠 [AI Memory Compression System](https://github.com/navaleprachi/AI_Memory_System)
> FastAPI · PostgreSQL · pgvector · OpenAI API · React · TypeScript

An LLM memory persistence system that compresses conversation history using OpenAI's GPT-4o-mini, reducing token usage by **85%** while preserving semantic recall. Built a full RAG pipeline with pgvector embeddings for intelligent context retrieval.

- Hierarchical summarization with configurable compression thresholds
- `/chat-with-debug` endpoint for inspecting memory state in real time
- React + TypeScript frontend with a Memory Inspector panel
